# 👥 KeenKeeper - Keep Your Friendships Alive

KeenKeeper is a professional friendship management application designed to help users track interactions and maintain long-term relationships.
---

## 🛠️ Technologies Used

- ⚡ **Framework:** Next.js (App Router)
- 🎨 **Styling:** Tailwind CSS & DaisyUI
- 📊 **Charts:** Recharts (Pie Chart for Analytics)
- 🔔 **Notifications:** React-Toastify
- ☁️ **Deployment:** Vercel

---

## ✨ Key Features

- 👥 **Friend Management**  
  Track days since last contact with smart status indicators (On-track, Overdue, etc.)

- 🕒 **Interactive Timeline**  
  Log and filter interactions like Calls, Texts, and Video calls

- 📈 **Friendship Analytics**  
  Visual insights using Pie Charts for better understanding of relationships

- 📱 **Responsive UI**  
  Fully optimized for Mobile, Tablet, and Desktop devices

- ⚠️ **Custom 404 & Loading**  
  Smooth user experience with custom error handling and loading animations
